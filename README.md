# Arvin CLI

## 简介

`Arvin CLI` 是一个基于 `Node.js` 的命令行工具，用于快速创建和管理项目。

## 安装

```bash
npm install -g arvin-cli
```

## 使用

```bash
arvin-cli create <project-name>
```

## 更新

```bash
arvin-cli update
```
